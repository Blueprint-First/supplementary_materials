Backbone & setup

Claude-Sonnet-4, temperature 0.7 (matches main paper Table I). ScienceWorld and ALFWorld use their official evaluation harnesses with no scoring modification; TravelPlanner numbers come from the official eval script.

Anonymity

Internal identifiers replaced with <ENTERPRISE>, <APP>, etc. If a reviewer spots any leak, please flag it in the rebuttal forum.

Reproducibility

Per-directory READMEs include the exact run commands and configuration. Each baseline run on the full 1000-task TravelPlanner takes ≈ 2 days; the 100-task subset is provided for reviewer-reproducible turnaround.
